## Project Overview

The Historical Fencing Portal is a centralized web platform for the European HEMA (Historical European Martial Arts) community, starting with Italy and expanding internationally.

### Core Modules

1. **Society Directory** - Public directory of fencing schools and associations
2. **Global Events Calendar** - Unified calendar for competitions, seminars, sparring sessions, and open training
3. **Erasmus Program** - Cross-society training visit program with verification system

### Key Features

- Multi-language support (Italian, English, German)
- SEO-optimized for discoverability
- Role-based access control (Visitor, User, Society Manager, Platform Admin)
- Verification system for participant safety
- Timezone-aware event management
- Free tier hosting (Vercel + Supabase)
